## Architecture Decisions

### Exercise Types
- `complete_sentence_input` — user types missing word(s)
- `complete_sentence_mcq` — user selects correct option from dropdown/buttons
- `reorder_translation` — user taps words to build sentence in order

### JSON Shapes
**answer (complete_sentence_input):** `["answer1", "alternative"]` or `"single_answer"`
**answer (complete_sentence_mcq):** `"correct_option"`
**answer (reorder_translation):** `["word1", "word2", "word3"]`
**options:** `["option1", "option2", "option3"]` (nullable)
**metadata.word_tokens:** see word-level translations spec

### Word Tokens (stored in exercise.metadata.word_tokens)
```json
[
  {
    "text": "preflight",
    "translation": "before the flight",
    "parts": [
      { "text": "pre", "translation": "before" },
      { "text": "flight", "translation": "the act of flying" }
    ]
  }
]
```

### Admin Auth
- Admin = `is_admin` boolean on users table
- Policy checks `$user->is_admin`

### Progress Tracking
- `user_course_progress` table: user_id, course_id, current_exercise_id, completed_exercise_ids (JSON)
